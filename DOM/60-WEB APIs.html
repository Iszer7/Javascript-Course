WEB APIs

Una API es una interfaz de programación de aplicaciones. Es un conjunto de rutinas que provee acceso a funciones de un determinado software

Manejo de eventos, formularios, consumo de datos con ajax y fetch, etc

DOM: Document Object Model

Es una interfase estandarizada, cada etiqueta de html es un nodo (elemento)

manipulacion de html mediante JS

BOM: Browser Object Model

No es una API estandar, actualmente los navegadores funcionan muy similar

detectar ubi del usuario, donde esta la ventana, manejo del historial, barra de dir.. etc

CSSOM: CSS Object Model

manipulacion de css mediante JS

control de eventos de animacion, etc
